# EKALOX Product Rules

- Dark theme
- Mobile first
- Reels-style home feed
- Each video represents one digital product
- CTA logic:
  - Paid product -> BUY NOW
  - Free product -> GET FREE
  - External app -> INSTALL
- MVP pages:
  - Home
  - Products
  - Upload
  - Profile
- Admin approval required before product is public
- Modular structure required
